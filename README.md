# Academic and Personal Projects

Welcome to my repository! This repository gathers a collection of projects I have completed as part of my studies and personal initiatives. Each folder contains a distinct project with its source files, resources, and, where available, detailed documentation. As you can probably see, there are many projects related to the UFC—I’m a huge fan of combat sports.

## Repository Structure
- `Dash_ufc/` : Creation of a Shiny dashboard about the UFC.  
- `Data_ufc_scrapy_projet/` : Creation of a dashboard using data scraped from the UFC website, stored in a MongoDB.
- `FUllstack-master/` : Creation of a data dashboard with an API for data transfer.
- `Scala-master/` : Development of an SQL Kernel Engine.
- `park&see/` : Simulation of a parking system.
- `synclead/` : Integration of a chatbot into a voice assistant. Unfortunately, I couldn’t include the LLM files I used because they were too large.

Each project includes a README in French, followed by an English version.

## Technologies Used
- **Languages:** Python, JavaScript, HTML/CSS, R, Scala, SQL, Docker, AWS  
- **Frameworks/Libraries:** Flask, React, Docker, Scrapy, Pandas, Hugging Face
